CSS Animations Examples

1. Continuous Animation: Pulsing Effect

Pulse!

This box continuously scales up slightly and emits a fading shadow, then returns to normal. It's a common effect for drawing attention.

2. Non-Continuous Animation: Entry Fade & Slide

This text fades in and slides up smoothly when the page loads, and then stops. Notice the delay and `forwards` fill mode.

Refresh the page to see this animation again.

3. Animation with Multiple Keyframes: Bounce Effect

Bounce!

This box demonstrates a classic "bounce" effect using multiple keyframes to simulate a realistic up-and-down motion.

4. CSS Sprite Animation (Steps)

This animation uses a sprite sheet and `steps()` to create a frame-by-frame walking animation. It cycles through 10 frames.

(Note: A real sprite sheet image is linked from an external source.)

5. Loading Spinner (Rotation)

A classic loading spinner created using `border-radius: 50%` and a simple `rotate` animation, useful for indicating background processes.

6. Text Typing Effect (Steps & Multiple Animations)

Hello, CSS Animations!

This effect simulates text being typed out, along with a blinking cursor. It combines two animations: one for `width` (using `steps`) and one for `border-color`.

7. Pulsing Background for a Section

This entire section has a subtle, continuous background color pulse. Notice how `alternate` makes the animation go back and forth.
